# 🫁 강철폐포부대

> **"유해물질의 위협으로부터 인류의 폐를 사수하라!"**  
> Pygame을 활용하여 제작된 한글 메디컬/헬스 교육 테마의 2D 디펜스 슈팅 게임입니다.  
> 플레이어는 귀여운 '폐포 병사'가 되어 먼지, 담배 연기, 고지방 음식 등 유해물질을 퇴치하고 혈중 산소포화도(SpO2)를 유지해야 합니다.

---

### 🚀 별도의 설치 없이 바로 플레이 가능! (Direct Play)
개발 및 빌드가 완료된 **단일 실행 파일(`강철폐포부대.exe`)**을 제공합니다.  
파이썬 설치 및 번잡한 환경 설정 없이 다운로드 후 더블 클릭만으로 게임을 즉시 즐기실 수 있습니다.

## ✨ 주요 특징 (Key Features)

1. **동적 폐포 부대 대열 시스템 (Squad Stacking)** 🪖
   * 아이템(브로콜리)을 획득할 때마다 폐포 병사가 최대 **30명**까지 증식합니다.
   * 병사가 늘어나면 가로 대열이 넓어져 동시 발사하는 총알의 양과 공격 범위가 비약적으로 상승하며, 10명이 넘어갈 때마다 윗줄로 겹쳐서 배치되는 자동 정렬 시스템을 구현했습니다.
2. **산소포화도(SpO2) 게이지 & 타임 어택** ⏱️
   * 게임 시작 시 산소포화도는 100%에서 매초 **5%**씩 감소합니다.
   * 적을 처치하면 산소포화도가 회복되며, 제한 시간 **60초** 동안 산소포화도를 **90% 이상**으로 유지한 채 생존해야 최종 승리합니다.
3. **시간 경과에 따른 난이도 자동 보정 (HP Scaling)** 📈
   * 시간이 흐를수록 스폰되는 적들의 체력(HP)이 초당 **0.15**의 속도로 강력해져, 후반부로 갈수록 더 박진감 넘치는 긴장감을 유도합니다.
4. **광역 필살기 '네불라이저 (Nebulizer)'** 🧪
   * 특정 시간(18초, 28초, 45초)에만 스폰되는 궁극기 아이템입니다.
   * 획득 시 전체 화면의 모든 적들에게 **50의 피해**를 주며, 푸른색 섬광 이펙트 및 데미지 플로팅 텍스트 연출로 짜릿한 타격감을 선사합니다.
5. **단계별 강력한 보스전** 👾
   * 게임 시작 20초 후 첫 보스가 등장하며, 이후 15초 간격으로 최대 3회 거대 보스가 나타납니다.
   * 각 보스는 순차적으로 **체력 100 / 200 / 300**을 보유하며, 등장 시 화면에 붉은 플래시와 화면 흔들림(Shake) 효과 및 경고음으로 위압감을 극대화했습니다.
6. **7장의 스토리 컷신 인트로** 🎥
   * 게임 시작 시 단순히 게임만 시작되는 것이 아니라, 7장의 스토리 일러스트(`story1.png` ~ `story7.png`) 컷신을 구현하여 게임의 배경 스토리와 몰입감을 한층 높였습니다.
7. **모니터 해상도 최적화 (Auto-Scaling)** 🖥️
   * 사용자의 모니터 해상도 정보를 읽어와 화면 비율을 벗어나지 않도록 **실시간 최적화 비율(80%)**로 게임 창 크기를 자동 조절합니다.

## 🕹️ 조작 방법 (Controls)

| 키 입력 | 기능 |
| :---: | :--- |
| `←` / `→` | 폐포 부대 좌우 이동 (이동 속도 기본값 4) |

## 👾 등장 적군 & 유해 물질 (Enemies)

시간이 흐를수록 기본 체력에 `소요 시간 * 0.15`만큼 추가 체력이 더해져 스폰됩니다.

| 적군 이미지 | 이름 | 기본 체력 (HP) | 처치 시 점수 | SpO2 회복량 |
| :---: | :---: | :---: | :---: | :---: |
| 💨 | **먼지** | 1 | 1점 | +1% |
| 🍔 | **고지방/고당분** | 2 | 6점 | +3% |
| 🚬 | **담배** | 4 | 10점 | +5% |
| 👾 | **보스 (1, 2, 3)** | 100 / 200 / 300 | 100점 | +10% |

## 🚀 플레이 및 실행 방법 (How to Play)

상황에 맞춰 아래의 두 가지 방법 중 하나를 선택해 플레이해 주세요.

### 방법 1. 실행 파일 다운로드 및 플레이 (가장 추천 ⭐)
1. 상단의 폴더 목록에서 **[강철폐포부대.exe](강철폐포부대.exe)** 파일을 다운로드합니다.
2. 다운로드받은 파일을 더블 클릭하여 실행하면 추가 설정 없이 게임이 즉시 시작됩니다.
   * *(보안 경고창이 나타날 경우, '추가 정보' 클릭 후 '실행'을 선택해 주시면 됩니다.)*

### 방법 2. 파이썬 소스코드로 실행 (개발자용 💻)
로컬 개발 환경에서 파이썬 코드를 직접 실행하거나 수정하고 싶을 때 사용합니다.

* **요구사항**: Python 3.8 이상, Pygame 라이브러리

```bash
# 레포지토리 클론 및 이동
git clone https://github.com/사용자이름/강철폐포부대.git
cd 강철폐포부대

# pygame 필수 패키지 설치
pip install pygame

# 게임 실행
python game.py
```

## 📦 실행 파일(.exe) 직접 빌드 방법 (PyInstaller)

만약 소스코드를 수정한 후 자신만의 `.exe` 실행 파일을 직접 빌드하고 싶다면 아래 명령어를 실행하십시오.  
`game.py` 내부의 `resource_path()` 헬퍼 함수가 패키징 환경에 맞게 경로를 보정해 줍니다.

```bash
# PyInstaller 설치
pip install pyinstaller

# Windows 환경에서 리소스를 포함해 단일 실행 파일로 빌드
pyinstaller --onefile --noconsole --add-data "image;image" --add-data "sound;sound" --add-data "font;font" game.py
```

*(참고: macOS/Linux 환경에서 빌드 시에는 `--add-data` 옵션의 구분자로 세미콜론 `;` 대신 콜론 `:`을 사용해야 합니다.)*
